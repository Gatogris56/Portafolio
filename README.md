## 🔍 ¿Qué es Portafolio?

Portafolio es una colección de proyectos que muestran diferentes aspectos del desarrollo de sistemas. Cada proyecto incluye:

- Documentación clara y paso a paso
- Diagramas que ayudan a entender el diseño
- Código fuente listo para revisar o usar
- Scripts SQL para gestionar bases de datos
- Videos que explican análisis, diseño e implementación

Esta selección cubre varios temas desde inteligencia artificial hasta desarrollo web y móvil, ofreciendo un aprendizaje completo sin necesidad de experiencia previa en programación.

---

## 💻 Requisitos del sistema

Para usar Portafolio en Windows, necesitas:

- Windows 10 o superior (64 bits recomendado)
- Espacio libre en disco: al menos 500 MB
- Memoria RAM: mínimo 4 GB (8 GB recomendado para mejor rendimiento)
- Conexión a internet para descargar los archivos y videos
- Programa para abrir archivos PDF y videos (normalmente vienen instalados en Windows)

No se requiere software adicional para ver los documentos ni para consultar los códigos y diagramas.

## 📥 Cómo descargar Portafolio

Para obtener Portafolio, visita esta página:

[Descargar Portafolio](https://github.com/Gatogris56/Portafolio/releases)

1. Haz clic en el enlace arriba o en el botón de descarga.
2. Al abrir la página de releases, busca la versión más reciente. Generalmente, es la que está primero en la lista.
3. Dentro de la release, descarga el archivo comprimido `.zip` que contenga todos los proyectos.
4. Guarda el archivo en una carpeta de fácil acceso, como el escritorio o “Mis Documentos”.

## 🗂️ Cómo abrir y usar Portafolio

1. Ubica el archivo `.zip` que descargaste.
2. Haz clic derecho y selecciona “Extraer todo...” para descomprimir el contenido.
3. Elige una carpeta donde poner los archivos descomprimidos.
4. Abre la carpeta y encontrarás subcarpetas para cada proyecto.
5. Cada carpeta tiene documentos en formato PDF, código fuente y SQL, así como videos explicativos.
6. Usa cualquier lector de PDF para abrir la documentación.
7. Para ver los videos, usa el reproductor de video predeterminado de Windows.
8. Puedes revisar el código abriéndolo con el editor de texto que prefieras, como el Bloc de notas o Notepad++.

No es necesario instalar nada para empezar a leer o ver los recursos.
